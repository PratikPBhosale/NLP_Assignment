# NLP Preprocessing and Text Classification

## Dataset
SMS Spam Detection Dataset (Kaggle)

## Objective
To apply NLP preprocessing techniques and build a classification model.

## Techniques Used
- Tokenization
- Stopword Removal
- Stemming
- Lemmatization
- TF-IDF Vectorization

## Models Used
- Naive Bayes
- Logistic Regression

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

## Results
Achieved ~97% accuracy on spam classification.
